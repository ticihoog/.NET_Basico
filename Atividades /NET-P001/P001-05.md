## **Excercício 05**

### **PROBLEMA:** Considere as variáveis int a = 5 e int b = 8. Escreva código para determinar se a é maior que b e exiba o resultado.

Segue código:

```
using System;

class Program
{
    static void Main()
    {
        int a = 5;
        int b = 8;

        if (a > b)
        {
            Console.WriteLine("a é maior que b");
        }
        else
        {
            Console.WriteLine("a não é maior que b");
        }
    }
}

```
